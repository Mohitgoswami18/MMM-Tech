# MMM-Tech — Gamified Financial Learning Platform

> Making financial literacy fun, interactive, and accessible for everyone (Age 10+)

---

## 🚀 Overview

It is a gamified financial education platform designed to help young users understand essential financial concepts through interactive modules and simulations.

The platform focuses on **learning by doing**, enabling users to make decisions, experience outcomes, and build real-world financial awareness in a safe virtual environment.

---

## 🎯 Target Audience

* 👦 Students aged **10+**
* 🧠 Anyone who wants to learn financial basics in a fun way

---

## ✨ Features

### 🧩 1. Needs vs Wants Module

An interactive learning module that helps users differentiate between **essential needs** and **optional wants**.

#### Key Highlights:

* Real-life scenario-based decisions
* Interactive UI for engagement
* Builds financial decision-making mindset

---

### 📈 2. Stock Market Simulator

A virtual trading environment where users can learn how the stock market works without real risk.

#### Key Highlights:

* 💰 Start with virtual money
* 📊 Simulated real-time stock price changes
* 🛒 Buy and sell stocks
* 📉 Track profit/loss
* 📚 Learn basic investing concepts
